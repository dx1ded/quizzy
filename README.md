# Quizzy App

The application provides with interactive quizzes in real time

#### Starting the app in development mode 

```bash
npm run dev
```

#### Building the app

```bash
npm run build
```

#### Linting the app

```bash
npm run lint
```

#### Linting + fixing

```bash
npm run lint-fix
```
